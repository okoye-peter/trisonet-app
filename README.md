# Trisonet

A full-stack application.

## Structure

- `/backend` - Node.js API server
- `/frontend` - Frontend application

## Setup Instructions

### Backend

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables by copying `.env.example` to `.env` (if available) and filling in the values.
4. Run the development server:
   ```bash
   npm run dev
   ```

### Frontend

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Environment Variables

Make sure to create a `.env` file in the root or the respective `backend` and `frontend` folders to connect to your database and other services. Ensure these files are **not checked into version control**.
